# **App Name**: AnonVerse

## Core Features:

- Anonymous Posting: Allow users to post messages anonymously.
- Feed Display: Display posts in a reverse chronological feed.
- Content Filtering: Filter displayed content based on keywords or user preferences, employing generative AI to identify content which violates those preferences. This filtering acts as a tool that ensures a safe and tailored user experience by dynamically assessing the relevance and safety of displayed posts.
- Real-time Updates: Provide real-time updates to the feed as new posts are made using Firebase.
- Database Migration: Migrate database from Supabase to Firebase to improve app database performance.
- User Authentication: Use Firebase Auth for simple authentication.

## Style Guidelines:

- Primary color: Dark Indigo (#3F51B5) to convey anonymity and depth.
- Background color: Very dark grayish blue (#283593) to complement the primary color and enhance the feeling of an alternative space.
- Accent color: Cyan (#00BCD4) to provide highlights and a modern, technological edge.
- Body and headline font: 'Inter' sans-serif for clear and accessible typography.
- Use minimalist icons to maintain the focus on content, with a technological theme.
- Use a card-based layout for posts to clearly separate content and improve readability.
- Subtle fade-in animations for new posts.
